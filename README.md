# Figmatt – TikTok UI

UI clone inspired by the [TikTok Figma design](https://www.figma.com/design/RQqrVL8OuP3l7B2e9ZNuRq/TikTok?node-id=1-4109). Built with React, TypeScript, Vite, and Tailwind CSS.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173). Use scroll or wheel to move between videos.

## Structure

- **Top**: “Following” / “For You” tabs
- **Main**: Full-height vertical feed with snap scroll
- **Right**: Profile avatar, like, comment, share, rotate
- **Bottom**: Username, caption, sound name
